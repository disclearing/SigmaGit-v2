# Introduction

Welcome to **Sigmagit** — a modern, self-hosted Git platform built for developers who want full control over their code. Think GitHub, but yours to own and run.

<Callout type="info">
  **Getting started?** Jump straight to the [Quick Start guide](/docs/quick-start) or follow the [Installation guide](/docs/installation) for a full setup.
</Callout>

## What is Sigmagit?

Sigmagit is an open-source Git hosting platform that provides everything you need to manage your code:

- **Git repository hosting** — Push, pull, clone with SSH and HTTPS
- **Issues & bug tracking** — Report bugs, request features, and track progress
- **Pull requests** — Code review workflows with comments and approvals
- **Discussions** — Community Q&A and threaded conversations
- **Project boards** — Kanban-style project management
- **Teams & organizations** — Collaborate with fine-grained permissions
- **Webhooks & API** — Integrate with your existing tools
- **Discord bot** — Get notifications and manage your repos from Discord

## Tech Stack

Sigmagit is built with modern, battle-tested technologies:

| Layer | Technology |
|-------|-----------|
| Frontend | React 19, TanStack Start, Tailwind CSS |
| API | Hono, Bun runtime |
| Database | PostgreSQL, Drizzle ORM |
| Git storage | isomorphic-git + AWS S3 |
| Auth | better-auth (email + passkeys) |
| Real-time | Redis + WebSockets |
| Mobile | Expo (React Native) |

## Why Self-Host?

- **Privacy** — Your code never leaves your infrastructure
- **Compliance** — Meet data sovereignty requirements
- **Cost** — No per-seat pricing for large teams
- **Customization** — Modify and extend to fit your workflow
- **No vendor lock-in** — Standard Git protocol, always portable

## Architecture

Sigmagit is a monorepo built with [Turbo](https://turbo.build), organized into:

```
sigmagit/
├── apps/
│   ├── web/        # Main web application
│   ├── api/        # REST API + Git protocol server
│   ├── mobile/     # iOS & Android app (Expo)
│   └── discord-bot/ # Discord integration
└── packages/
    ├── db/         # Database schema & migrations
    ├── hooks/      # React Query hooks
    └── lib/        # Shared utilities
```
