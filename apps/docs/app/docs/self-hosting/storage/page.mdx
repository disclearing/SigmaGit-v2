# Storage Setup

Sigmagit stores Git repository data in **S3-compatible object storage**.

## Supported Providers

| Provider | Notes |
|----------|-------|
| AWS S3 | Production-ready, highly available |
| MinIO | Self-hosted, excellent for privacy |
| Cloudflare R2 | No egress fees, S3-compatible |
| Backblaze B2 | Affordable, S3-compatible |
| Wasabi | Cost-effective S3-compatible |

## AWS S3

```bash
S3_ENDPOINT=https://s3.amazonaws.com
S3_BUCKET=my-sigmagit-repos
S3_ACCESS_KEY=AKIAIOSFODNN7EXAMPLE
S3_SECRET_KEY=wJalrXUtnFEMI/K7MDENG/bPxRfiCYEXAMPLEKEY
S3_REGION=us-east-1
```

Create the bucket:

```bash
aws s3 mb s3://my-sigmagit-repos --region us-east-1
```

## MinIO (Self-hosted)

```bash
# Start MinIO with Docker
docker run -d \
  --name minio \
  -p 9000:9000 \
  -p 9001:9001 \
  -e MINIO_ROOT_USER=admin \
  -e MINIO_ROOT_PASSWORD=password123 \
  -v /data/minio:/data \
  minio/minio server /data --console-address ":9001"
```

```bash
S3_ENDPOINT=http://localhost:9000
S3_BUCKET=sigmagit-repos
S3_ACCESS_KEY=admin
S3_SECRET_KEY=password123
S3_REGION=us-east-1
S3_FORCE_PATH_STYLE=true
```

Create the bucket via MinIO Console at `http://localhost:9001`.

## Cloudflare R2

```bash
S3_ENDPOINT=https://<account_id>.r2.cloudflarestorage.com
S3_BUCKET=sigmagit-repos
S3_ACCESS_KEY=<r2_access_key_id>
S3_SECRET_KEY=<r2_secret_access_key>
S3_REGION=auto
```

<Callout type="info">
  Cloudflare R2 has zero egress fees, making it ideal for high-traffic instances.
</Callout>

## Bucket Configuration

### Required Permissions

The IAM user or service account needs these permissions:

```json
{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Effect": "Allow",
      "Action": [
        "s3:GetObject",
        "s3:PutObject",
        "s3:DeleteObject",
        "s3:ListBucket"
      ],
      "Resource": [
        "arn:aws:s3:::my-sigmagit-repos",
        "arn:aws:s3:::my-sigmagit-repos/*"
      ]
    }
  ]
}
```

## Data Structure

Sigmagit organizes data in the bucket as:

```
<bucket>/
└── repos/
    └── <username>/
        └── <repo>/
            ├── HEAD
            ├── objects/
            ├── refs/
            └── ...
```
