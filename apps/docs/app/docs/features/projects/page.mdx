# Projects

Projects are Kanban-style boards for organizing issues and pull requests.

## Creating a Project

1. Go to your repository's **Projects** tab
2. Click **New project**
3. Give it a name and optional description

## Columns

Projects have customizable columns representing workflow stages. Default columns:

- **To Do** — Work not yet started
- **In Progress** — Work currently being done
- **Done** — Completed work

## Cards

Each card on the board represents either:

- An **Issue** from the repository
- A **Pull Request** from the repository
- A **Note** — a free-form text card

### Drag & Drop

Cards can be dragged between columns to update their status. Cards within a column can also be reordered.
