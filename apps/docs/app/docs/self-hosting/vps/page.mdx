# VPS Deployment

This guide walks through deploying Sigmagit on a VPS (e.g., DigitalOcean, Linode, Hetzner).

## Server Setup

```bash
# Update the system
sudo apt update && sudo apt upgrade -y

# Install required packages
sudo apt install -y curl git nginx certbot python3-certbot-nginx
```

## Install Bun

```bash
curl -fsSL https://bun.sh/install | bash
source ~/.bashrc
bun --version
```

## Install PostgreSQL

```bash
sudo apt install -y postgresql postgresql-contrib

# Start and enable
sudo systemctl start postgresql
sudo systemctl enable postgresql

# Create database and user
sudo -u postgres psql <<EOF
CREATE USER sigmagit WITH PASSWORD 'your-password';
CREATE DATABASE sigmagit OWNER sigmagit;
GRANT ALL PRIVILEGES ON DATABASE sigmagit TO sigmagit;
EOF
```

## Install Redis

```bash
sudo apt install -y redis-server
sudo systemctl start redis
sudo systemctl enable redis
```

## Deploy Sigmagit

```bash
# Create app user
sudo useradd -m -s /bin/bash sigmagit
sudo su - sigmagit

# Clone the repository
git clone https://github.com/sigmagit/sigmagit.git
cd sigmagit

# Install dependencies
bun install

# Configure environment
cp .env.example .env
nano .env  # Fill in your values

# Build
bun run build

# Run migrations
bun run db:migrate
```

## Systemd Services

Create service files for automatic startup:

```ini title="/etc/systemd/system/sigmagit-api.service"
[Unit]
Description=Sigmagit API
After=network.target postgresql.service redis.service

[Service]
Type=simple
User=sigmagit
WorkingDirectory=/home/sigmagit/sigmagit/apps/api
ExecStart=/home/sigmagit/.bun/bin/bun run start
Restart=always
RestartSec=5
EnvironmentFile=/home/sigmagit/sigmagit/.env

[Install]
WantedBy=multi-user.target
```

```ini title="/etc/systemd/system/sigmagit-web.service"
[Unit]
Description=Sigmagit Web
After=network.target sigmagit-api.service

[Service]
Type=simple
User=sigmagit
WorkingDirectory=/home/sigmagit/sigmagit/apps/web
ExecStart=/home/sigmagit/.bun/bin/bun run start
Restart=always
RestartSec=5
EnvironmentFile=/home/sigmagit/sigmagit/.env

[Install]
WantedBy=multi-user.target
```

```bash
sudo systemctl daemon-reload
sudo systemctl enable sigmagit-api sigmagit-web
sudo systemctl start sigmagit-api sigmagit-web
```

## Configure Nginx

```nginx title="/etc/nginx/sites-available/sigmagit"
server {
    server_name sigmagit.example.com;

    location / {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_cache_bypass $http_upgrade;
    }

    location /api/ {
        proxy_pass http://localhost:3001/;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

```bash
sudo ln -s /etc/nginx/sites-available/sigmagit /etc/nginx/sites-enabled/
sudo nginx -t && sudo systemctl reload nginx

# Get SSL certificate
sudo certbot --nginx -d sigmagit.example.com
```
