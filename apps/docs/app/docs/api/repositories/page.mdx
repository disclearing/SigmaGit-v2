# Repositories API

## List User Repositories

```http
GET /api/repositories
Authorization: Bearer <token>
```

**Query Parameters:**

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `page` | number | `1` | Page number |
| `per_page` | number | `30` | Results per page (max 100) |
| `visibility` | string | — | Filter by `public` or `private` |
| `sort` | string | `updated` | Sort by `created`, `updated`, `name` |

**Response:**

```json
{
  "repositories": [
    {
      "id": "repo_abc",
      "name": "my-project",
      "fullName": "username/my-project",
      "description": "My project description",
      "visibility": "public",
      "defaultBranch": "main",
      "stars": 42,
      "forks": 5,
      "language": "TypeScript",
      "updatedAt": "2024-01-15T10:30:00Z"
    }
  ],
  "total": 15
}
```

## Get a Repository

```http
GET /api/repos/:owner/:repo
Authorization: Bearer <token>
```

## Create a Repository

```http
POST /api/repositories
Authorization: Bearer <token>
Content-Type: application/json

{
  "name": "my-new-repo",
  "description": "Optional description",
  "visibility": "public",
  "defaultBranch": "main",
  "initializeWithReadme": true
}
```

## Update a Repository

```http
PATCH /api/repos/:owner/:repo
Authorization: Bearer <token>
Content-Type: application/json

{
  "description": "Updated description",
  "visibility": "private"
}
```

## Delete a Repository

```http
DELETE /api/repos/:owner/:repo
Authorization: Bearer <token>
```

<Callout type="danger">
  **This action is irreversible.** All data including issues, PRs, and the Git history will be permanently deleted.
</Callout>

## List Branches

```http
GET /api/repos/:owner/:repo/branches
```

## Get File Contents

```http
GET /api/repos/:owner/:repo/contents/:path?ref=main
```

## Get Commits

```http
GET /api/repos/:owner/:repo/commits?branch=main&page=1
```
