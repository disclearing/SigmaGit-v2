# Discord API

API endpoints for Discord bot integration, account linking, and webhook notifications.

## Base URL

```
https://your-sigmagit-instance.com/api/discord
```

## Authentication

Discord API endpoints require authentication via Bearer token. Some endpoints may be called by the Discord bot using internal authentication.

## Account Linking

### Generate Link Token

Generate a verification token for linking a Discord account to a Sigmagit account.

```http
POST /api/discord/link/generate
Content-Type: application/json
```

**Request Body:**
```json
{
  "discordId": "123456789012345678",
  "sigmagitEmail": "user@example.com"
}
```

**Parameters:**
- `discordId` (required) — Discord user ID
- `sigmagitEmail` (optional) — Sigmagit account email

**Response:**
```json
{
  "success": true,
  "token": "abc123def456..."
}
```

**Errors:**
- `400` — Discord ID is required
- `404` — User not found with that email
- `500` — Internal server error

### Verify Link Token

Verify a link token and complete account linking.

```http
POST /api/discord/link/verify
Content-Type: application/json
```

**Request Body:**
```json
{
  "token": "abc123def456...",
  "sigmagitUserId": "user-id"
}
```

**Parameters:**
- `token` (required) — Verification token from generate endpoint
- `sigmagitUserId` (required) — Sigmagit user ID

**Response:**
```json
{
  "success": true,
  "linked": true,
  "user": {
    "id": "user-id",
    "username": "username",
    "email": "user@example.com"
  }
}
```

**Errors:**
- `400` — Invalid or expired token, token already used, or token mismatch
- `404` — User not found
- `500` — Internal server error

### Get Link Status

Get the link status for a Discord account.

```http
GET /api/discord/link/status/:discordId
```

**Parameters:**
- `discordId` (path) — Discord user ID

**Response (Linked):**
```json
{
  "linked": true,
  "verified": true,
  "user": {
    "id": "user-id",
    "username": "username",
    "name": "User Name",
    "avatarUrl": "https://..."
  },
  "linkedAt": "2024-01-01T00:00:00Z"
}
```

**Response (Not Linked):**
```json
{
  "linked": false
}
```

**Errors:**
- `400` — Discord ID is required
- `500` — Internal server error

### Unlink Account

Unlink a Discord account from Sigmagit.

```http
POST /api/discord/link/unlink
Content-Type: application/json
```

**Request Body:**
```json
{
  "discordId": "123456789012345678"
}
```

**Parameters:**
- `discordId` (required) — Discord user ID

**Response:**
```json
{
  "success": true,
  "unlinked": true
}
```

**Errors:**
- `400` — Discord ID is required
- `404` — No link found for this Discord ID
- `500` — Internal server error

### Relink Account

Relink a Discord account to a different Sigmagit account.

```http
POST /api/discord/link/relink
Content-Type: application/json
```

**Request Body:**
```json
{
  "discordId": "123456789012345678",
  "sigmagitEmail": "user@example.com"
}
```

**Parameters:**
- `discordId` (required) — Discord user ID
- `sigmagitEmail` (required) — New Sigmagit account email

**Response:**
```json
{
  "success": true,
  "relinked": true,
  "user": {
    "id": "user-id",
    "username": "username",
    "email": "user@example.com"
  }
}
```

**Errors:**
- `400` — Discord ID and email are required
- `404` — User not found or no existing link found
- `500` — Internal server error

## Webhooks

### Send Discord Webhook

Send a webhook notification to Discord.

```http
POST /api/webhooks/discord
Content-Type: application/json
```

**Request Body:**
```json
{
  "type": "issue",
  "repository": {
    "owner": "username",
    "name": "repo-name"
  },
  "data": {
    "action": "opened",
    "number": 123,
    "title": "Issue title",
    "body": "Issue description",
    "state": "open",
    "author": {
      "username": "author",
      "name": "Author Name"
    },
    "createdAt": "2024-01-01T00:00:00Z"
  }
}
```

**Event Types:**
- `issue` — Issue events (opened, closed, edited)
- `pull_request` — Pull request events (opened, closed, merged)
- `push` — Push events (new commits)
- `star` — Star events (added, removed)
- `fork` — Fork events

**Response:**
```json
{
  "success": true
}
```

**Errors:**
- `400` — Invalid webhook payload
- `501` — Discord webhook not configured
- `500` — Internal server error

### Test Discord Webhook

Test the Discord webhook configuration.

```http
GET /api/webhooks/discord/test
```

**Response:**
```json
{
  "success": true,
  "message": "Test webhook sent"
}
```

**Errors:**
- `501` — Discord webhook not configured
- `500` — Internal server error

## Rate Limiting

Discord API endpoints follow the same rate limiting as other API endpoints:

- **1000 requests per hour** per authenticated user
- Rate limit headers included in responses

## Error Codes

| Code | Meaning |
|------|---------|
| `400` | Bad Request — invalid input |
| `401` | Unauthorized — missing or invalid token |
| `404` | Not Found — resource doesn't exist |
| `500` | Internal Server Error |
| `501` | Not Implemented — feature not configured |

## Examples

### Link Account (cURL)

```bash
# Generate token
curl -X POST https://sigmagit.com/api/discord/link/generate \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <token>" \
  -d '{
    "discordId": "123456789012345678",
    "sigmagitEmail": "user@example.com"
  }'

# Verify token
curl -X POST https://sigmagit.com/api/discord/link/verify \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <token>" \
  -d '{
    "token": "abc123...",
    "sigmagitUserId": "user-id"
  }'
```

### Check Link Status

```bash
curl -X GET https://sigmagit.com/api/discord/link/status/123456789012345678 \
  -H "Authorization: Bearer <token>"
```

### Send Webhook

```bash
curl -X POST https://sigmagit.com/api/webhooks/discord \
  -H "Content-Type: application/json" \
  -d '{
    "type": "issue",
    "repository": {
      "owner": "username",
      "name": "repo"
    },
    "data": {
      "action": "opened",
      "number": 1,

    }
  }'
```
