# Issues API

## List Issues

```http
GET /api/repos/:owner/:repo/issues
Authorization: Bearer <token>
```

**Query Parameters:**

| Parameter | Type | Description |
|-----------|------|-------------|
| `state` | string | `open` (default) or `closed` |
| `label` | string | Filter by label name |
| `assignee` | string | Filter by assignee username |
| `page` | number | Page number |
| `per_page` | number | Results per page |

## Get an Issue

```http
GET /api/repos/:owner/:repo/issues/:number
```

**Response:**

```json
{
  "id": "issue_abc",
  "number": 42,
  "title": "Bug: Something is broken",
  "body": "Description of the bug...",
  "state": "open",
  "author": {
    "username": "johndoe",
    "avatarUrl": "https://..."
  },
  "labels": [
    { "id": "lbl_1", "name": "bug", "color": "ef4444" }
  ],
  "assignees": [],
  "commentCount": 3,
  "createdAt": "2024-01-10T09:00:00Z",
  "updatedAt": "2024-01-15T14:30:00Z"
}
```

## Create an Issue

```http
POST /api/repos/:owner/:repo/issues
Authorization: Bearer <token>
Content-Type: application/json

{
  "title": "New bug report",
  "body": "## Steps to Reproduce\n\n1. ...",
  "labels": ["bug"],
  "assignees": ["username"]
}
```

## Update an Issue

```http
PATCH /api/repos/:owner/:repo/issues/:number
Authorization: Bearer <token>
Content-Type: application/json

{
  "state": "closed",
  "title": "Updated title"
}
```

## Add a Comment

```http
POST /api/repos/:owner/:repo/issues/:number/comments
Authorization: Bearer <token>
Content-Type: application/json

{
  "body": "Thanks for reporting this! We'll look into it."
}
```

## List Comments

```http
GET /api/repos/:owner/:repo/issues/:number/comments
```

## Lock an Issue

```http
PUT /api/repos/:owner/:repo/issues/:number/lock
Authorization: Bearer <token>
```

## Unlock an Issue

```http
DELETE /api/repos/:owner/:repo/issues/:number/lock
Authorization: Bearer <token>
```
