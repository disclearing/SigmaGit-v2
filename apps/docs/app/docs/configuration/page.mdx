# Configuration

Sigmagit is configured via environment variables. Copy `.env.example` to `.env` and edit the values.

## Core Settings

| Variable | Required | Default | Description |
|----------|----------|---------|-------------|
| `APP_URL` | ✅ | — | Public URL of the web app |
| `API_URL` | ✅ | — | Public URL of the API server |
| `BETTER_AUTH_SECRET` | ✅ | — | Secret key for auth (min 32 chars) |

Generate a secure secret:

```bash
openssl rand -hex 32
```

## Database

| Variable | Required | Default | Description |
|----------|----------|---------|-------------|
| `DATABASE_URL` | ✅ | — | PostgreSQL connection string |

```bash
DATABASE_URL=postgresql://user:password@localhost:5432/sigmagit
```

## Storage (S3)

Git repository data is stored in S3-compatible object storage.

| Variable | Required | Default | Description |
|----------|----------|---------|-------------|
| `S3_ENDPOINT` | ✅ | — | S3 endpoint URL |
| `S3_BUCKET` | ✅ | `sigmagit-repos` | S3 bucket name |
| `S3_ACCESS_KEY` | ✅ | — | S3 access key ID |
| `S3_SECRET_KEY` | ✅ | — | S3 secret access key |
| `S3_REGION` | ✅ | `us-east-1` | S3 region |
| `S3_FORCE_PATH_STYLE` | ❌ | `false` | Use path-style URLs (for MinIO) |

### AWS S3

```bash
S3_ENDPOINT=https://s3.amazonaws.com
S3_BUCKET=my-sigmagit-repos
S3_ACCESS_KEY=AKIAIOSFODNN7EXAMPLE
S3_SECRET_KEY=wJalrXUtnFEMI/K7MDENG/bPxRfiCYEXAMPLEKEY
S3_REGION=us-east-1
```

### MinIO (Self-hosted)

```bash
S3_ENDPOINT=http://localhost:9000
S3_BUCKET=sigmagit-repos
S3_ACCESS_KEY=minioadmin
S3_SECRET_KEY=minioadmin
S3_REGION=us-east-1
S3_FORCE_PATH_STYLE=true
```

## Redis (Optional)

Redis enables real-time features (WebSockets, live notifications).

| Variable | Required | Default | Description |
|----------|----------|---------|-------------|
| `REDIS_URL` | ❌ | — | Redis connection string |

```bash
REDIS_URL=redis://localhost:6379
# With auth:
REDIS_URL=redis://:password@localhost:6379
```

<Callout type="warning">
  Without Redis, real-time features like live notification updates are disabled. The app works fine without it.
</Callout>

## Discord Bot (Optional)

| Variable | Required | Default | Description |
|----------|----------|---------|-------------|
| `DISCORD_BOT_TOKEN` | ❌ | — | Discord bot token |
| `DISCORD_CLIENT_ID` | ❌ | — | Discord application client ID |
| `DISCORD_GUILD_ID` | ❌ | — | Guild ID for slash command registration |
| `DISCORD_WEBHOOK_URL` | ❌ | — | Discord webhook URL for notifications |
| `WEBHOOK_SECRET` | ❌ | — | Secret key for webhook signature verification |
| `PUBLIC_WEBHOOK_URL` | ❌ | — | Public URL for webhook endpoints |

### Discord Webhook Setup

1. Go to your Discord server settings
2. Navigate to Integrations → Webhooks
3. Create a new webhook
4. Copy the webhook URL
5. Set `DISCORD_WEBHOOK_URL` in your environment

```bash
DISCORD_WEBHOOK_URL=https://discord.com/api/webhooks/YOUR_WEBHOOK_ID/YOUR_WEBHOOK_TOKEN
WEBHOOK_SECRET=your-secret-key-for-signature-verification
PUBLIC_WEBHOOK_URL=https://your-domain.com/api/webhooks/discord
```
